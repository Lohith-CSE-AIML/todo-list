# 📝 To-Do List App

A simple To-Do List web application built using **HTML, CSS, and JavaScript**.

## 🚀 Features

* Add new tasks
* Delete tasks
* Mark tasks as completed
* Simple and clean UI

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (DOM Manipulation)
* Local Storage
